Vous voulez commencer à coder avec Kotlin et vous ne savez pas quels outils utiliser ? Voici les deux seule choses dont vous avez besoin.

Les outils

Imaginez un instant qu'on vous donne ca:

Et qu'on vous demande d'aller front et de vous battre contre une horde de mercenaires armés jusqu'aux dents. Vous pensez bien qu'il va y avoir un problème...

Quand vous programmez, c'est pareil. Avant de se lancer à l'attaque d'un problème, il faut savoir bien s'équiper. Quel que soit votre systeme et le language de programmation que vous apprenez, il va falloir se munir des bons outils pour programmer. Nous allons donc commencer par installer les armes dont nous avons besoin pour mener à bien notre mission.

IntelliJ

IntelliJ est un IDE ou EDI en francais. Le principe d'un IDE est de concentrer tous les outils nécessaires à l'écriture d'un programme en un seul editeur. Cette caracteristique va notemment nous éviter d'avoir à utiliser des lignes de commande.

Il existe deux versions de cet outil. Une version gratuite (Community) largement suffisante pour nos besoins et une version commerciale payante (Ultimate) dont les avantages sont intéressants uniquement quand programmer devient votre metier.

Un JDK

Le JDK est un ensemble d'outils qui vont être utilisés par IntelliJ pour construire et executer nos programmes. Nous n'allons pas utiliser ces outils directement mais ils sont nécessaires pour la suite.

L'installation

Windows

  1. JDK

    Vous pouvez téléchqrger un JDK ici. Suivez le lien qui va vers le telechargement du JDK.

    Le JRE est suffisant pour executer les programmes que l'on va creer mais ne permet pas de les construire. Choisissez bien un JDK.

    Parmis les choix possibles, nous voulons un JDK pour windows. Il en existe deux versions: une pour Windows 64bits et 32bits. Si vous ne savez pas quelle version vous avez, vous pouvez prendre la version 32bits qui fonctionnera aussi sur les systemes 64bits. Si vous souhaitez verifier quelles version vous avez, aller dans Panneau de configuration > Système et securite > Système

    Installer le JDK. Les options par défaut conviendront parfaitement.

  2. IntelliJ

    Vous pouvez téléchqrger IntelliJ depuis le site officiel de JetBrains. Choisissez ensuite la Community Edition.

    Vous pouvez maintenant l'installer. Toutes les options par défaut conviennent pour l'usage que nous allons en faire.

Linux

  1. JDK

    Je vous invite a vous référer a votre gestionnaire de packages et à chercher les paquets du nom de Open JDK. Au moment ou j'écris ces lignes la version stable officielle est le JDK 8. Dans le cas des systèmes basés sur Debian, le paquet se nomme openjdk-8-jdk. Il existe aussi un packet openjdk-8-jre qui ne contient que les outils pour éxécuter (et non pas créer) un programme et il n'est pas suffisant de faire du développement.

  2. IntelliJ

    L'installation d'IntelliJ se passe comme avec la version Windows. Une fois le fichier télécharge depuis le site officiel de JetBrains, il vous suffit de le décompresser. Dans le dossier, vous trouverez un script pour lancer l'IDE. En ligne de commande, ca donne: ./idea-IC-171.4694.23/bin/idea.sh
