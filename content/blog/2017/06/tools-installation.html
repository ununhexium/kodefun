Les outils

Quel que soit votre systeme et le language de programmation que vous apprenez, il va falloir se munir des bons outils pour programmer.

IntelliJ

IntelliJ est un IDE ou EDI en francais. Le principe d'un IDE est de concentrer tous les outils necessaire a l'ecriture d'un programme en un seul editeur. Cette caracteristique va notemment nous eviter d'avoir a utiliser des lignes de commande.

Il existe deux versions de cet outil. Une version gratuite (Community) largement suffisante pour nos besoins et une version commerciale payante (Ultimate) dont les avantages sont interessants uniquement quand programmer devient votre metier.

Un JDK

Le JDK est un ensemble d'outils qui vont etre utilises par IntelliJ pour executer nos programmes. Nous n'allons pas utiliser ces outils directement mais ils sont necessaires pour la suite.

L'installation

Windows

  1. JDK

    Vous pouvez telecharger un JDK ici. Suivez le lien qui va vers le telechargement du JDK.

    Le JRE est suffisant pour executer les programmes que l'on va creer mais ne permet pas de les construire. Choisissez bien un JDK.

    Parmis les choix possibles, nous voulons un JDK pour windows. Il en existe deux versions: une pour Windows 64bits et 32bits. Si vous ne savez pas quelle version vous avez, vous pouvez prendre la version 32bits qui fonctionnera aussi sur les systemes 64bits. Si vous souhaitez verifier quelles version vous avez, aller dans Panneau de configuration > Systeme et securite > System

    Installer le JDK. Les options par defaut conviendront parfaitement.

  2. IntelliJ

    Vous pouvez telecharger IntelliJ depuis le site officiel de JetBrains. Choisissez ensuite la Community Edition.

    Vous pouvez maintenant l'installer. Toutes les options par defaut conviennent pour l'usage que nous allons en faire.

Linux

Je vous invite a vous referer a votre gestionnaire de packages et a chercher les paquets du nom de Open JDK. Au moment ou j'ecris ces lignes la version stable officielle est le JDK 8. Dans le cas des systeme bases sur Debian, le paquet se nomme openjdk-8-jdk. Il existe aussi un packet openjdk-8-jre qui ne contient que les outils pour executer (et non pas creer) un programme et il n'est pas suffisant de faire du developpement avec.
