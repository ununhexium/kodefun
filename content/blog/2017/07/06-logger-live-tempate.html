Logger live template

En java, il arrive souvent de vouloir rajouter un logger a une classe. Le probleme de cet ajout c'est qu'il prend une ligne complete et du temps, meme avec la completion automatique.

Solution: ce live template.

private static final Logger LOG =
  java.util.logging.Logger.getLogger($CLASS$.class.getCanonicalName());
$END$

The value for the CLASS variable:

className()

Il n'y a plus qu'a taper log et Entree deux foix dans le code source pour declarer ce logger.
