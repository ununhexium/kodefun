Les scripts Kotlin

Les scripts Kotlin sont des programmes ecrits avec la meme syntaxe qu'on programme Kotlin. Leur avantage est qu'il permet d'ecrire du code comme dans le REPL.

Comment creer un script Kotlin?

L'extension des fichiers Kotlin script est .kts. IntelliJ n'a pas de raccourci pour creer ce type de fichier directement et il va falloir creer un fichier generique puis lui attribuer l'extension kts.

Une fois le fichier creer, on peut y ecrire le code que l'on veut et l'executer.
